## Run Locally

Clone the project

```bash
  git clone https://github.com/asarelcastellanos/BLOCKTALK
```

Go to the project directory

```bash
  cd BLOCKTALK
```

Install dependencies

```bash
  yarn install
```

Run on IOS

```bash
  expo start i
```

Run on ANDRIOD

```bash
  expo start a
```
